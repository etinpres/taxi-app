# 🚕 고도비만 택시장부

택시 운전 수입/지출 관리 웹앱입니다.

## 🚀 바로 사용하기

이 앱은 웹 브라우저에서 바로 실행됩니다!

### 📱 모바일에 설치하기

#### iPhone (iOS)
1. Safari로 이 페이지 열기
2. 하단 공유 버튼 (□↑) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭
5. 완료!

#### Android
1. Chrome으로 이 페이지 열기
2. "홈 화면에 추가" 팝업에서 "설치" 탭
3. 완료!

## ✨ 주요 기능

- ✅ 수입/지출 내역 관리
- ✅ 카테고리 커스터마이징
- ✅ 캘린더 뷰로 한눈에 확인
- ✅ 월별 요약 통계
- ✅ 내역 수정/삭제
- ✅ 로컬 저장 (데이터 보호)
- ✅ 오프라인 지원
- ✅ 모바일 최적화

## 💾 데이터 저장

모든 데이터는 브라우저에 안전하게 저장됩니다.
- 인터넷 연결 없이도 작동
- 개인 정보 보호 (서버에 전송 안 됨)

## 🔧 기술 스택

- React 18
- Tailwind CSS
- Lucide Icons
- Service Worker (PWA)
- LocalStorage
